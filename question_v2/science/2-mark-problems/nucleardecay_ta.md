${}^{235}_{92}U$ ஒரு ஆல்பா சிதைவும் மற்றும் ஒரு பீட்டா சிதைவும் அடைகிறது. இறுதியில் உருவாகும் உட்கருவில் உள்ள நியூட்ரான்களின் எண்ணிக்கையை கணக்கிடுக.

```markdown
ஆல்பா சிதைவு:
${}^{235}_{92}U \rightarrow {}^{231}_{90}X + {}^{4}_{2}He$

பீட்டா சிதைவு:
${}^{231}_{90}X \rightarrow {}^{231}_{91}Y + {}^{0}_{-1}e$

இறுதி அணுக்கரு:
நிறை எண் = 231
அணு எண் = 91

நியூட்ரான்களின் எண்ணிக்கை:
= நிறை எண் - அணு எண்
= 231 - 91
= 140

ஆகவே:
நியூட்ரான்களின் எண்ணிக்கை = 140
```
