  Which is used to edit programs?

```markdown
In programming environments, the script editor (or code editor) is the specific interface where a user writes, modifies, and manages the code or scripts. While Inkscape is a vector graphics tool and the stage is for viewing output, the script editor is the primary tool for editing the logic of the program.
```
