 Where you will create category of blocks?

```markdown
In Scratch, the Block Menu (located on the far left) contains the different categories of blocks such as Motion, Looks, Sound, and Events. Clicking on a category in the Block Menu filters the Block Palette to show only the blocks related to that specific category. Users can also create their own categories by using the 'My Blocks' or 'Extensions' features found there.
```
