  Which is used to build scripts?

```markdown
In Scratch, the Script Area is the place where you drag and connect blocks together to build scripts. The Block Palette contains the available blocks, the Stage is where the animation is displayed, and the Sprite is the character or object that follows the instructions in the script.
```
