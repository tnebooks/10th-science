---
choices:
  - CH≡CH < CH4 < CH2 = CH2
  - CH ≡ CH < CH2 = CH2 < CH4
  - CH4 < CH ≡ CH < CH2 = CH2
answers:
  - CH4 < CH2 = CH2 < CH ≡ CH
explanation: >-
  ஹைட்ரோகார்பன்களின் வினைதிறன் (reactivity), கார்பன்-கார்பன் பிணைப்புகளின் 
  வகையைப் பொறுத்தது. அல்கேன்கள் ($CH_4$) ஒற்றை பிணைப்புகளை மட்டும் 
  கொண்டுள்ளதால் வினைதிறன் குறைந்தவை. அல்கீன்கள் ($CH_2=CH_2$) இரட்டைப் 
  பிணைப்பைக் கொண்டுள்ளதால் அதிக வினைதிறன் கொண்டவை. அல்கைன்கள் ($CH \equiv CH$) 
  முப்பிணைப்பைக் கொண்டுள்ளதால் (இரண்டு $\pi$ பிணைப்புகள்), இந்த வரிசையில் 
  மிக அதிக வினைதிறன் கொண்டவையாக உள்ளன.
