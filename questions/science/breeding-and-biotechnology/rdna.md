rDNA is a


```markdown
  rDNA (recombinant DNA) is formed when a vector DNA (plasmid, bacteriophage,
  etc.) is joined with desired DNA/gene of interest using DNA ligase. This
  recombinant molecule can then be introduced into a host cell for cloning or
  expression.
```
