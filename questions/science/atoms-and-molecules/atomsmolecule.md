  - mono atomic molecule
  - hetero atomic molecule
  - poly atomic molecule
answers:
  - homo atomic molecule
explanation: >-
  Molecules consist of two or more atoms chemically bonded together. If these 
  atoms are of the same element (e.g., $H_2$, $O_2$, $O_3$), the molecule 
  is called a homoatomic molecule. If the atoms are from different elements 
  (e.g., $H_2O$), it is called a heteroatomic molecule.
