Where should an object be placed so that a real and inverted image of same size is obtained by a convex lens?


```markdown
  For a convex lens, when an object is placed at the center of curvature (2f),
  the light rays refract to form an image on the opposite side at (2f). This
  image is real, inverted, and exactly the same size as the original object.
```
