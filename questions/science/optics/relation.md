If 𝑉_𝐵, 𝑉_𝐺,𝑉_𝑅 are the velocities of blue, green, and red light in a glass prism, which relation is correct?

```markdown
  Blue light has the shortest wavelength → highest refractive index → slowest
  speed, while red light has the longest wavelength → lowest refractive index →
  fastest speed.
```
