  - "V_B = V_G = V_R"
  - "V_B > V_G > V_R"
  - "V_B < V_G < V_R"
  - "V_B < V_G > V_R"
answer:
  - "V_B < V_G < V_R"
explanation: "Blue light has the shortest wavelength → highest refractive index → slowest speed, while red light has the longest wavelength → lowest refractive index → fastest speed."
