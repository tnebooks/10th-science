The value of Avogadro number

```markdown
  The Avogadro number ($N_A$) represents the number of constituent particles 
  (usually atoms or molecules) that are contained in one mole of a substance. 
  It is a constant value approximately equal to $6.023 \times 10^{23} \text{ mol}^{-1}$.
```
