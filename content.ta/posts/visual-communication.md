## மதிப்பீடு

### I. சரியான விடையைத் தேர்ந்தெடுக்கவும்

1. அனிமேஷன் பயன்படும் தேனிப்பொருள் எது?

   - a) Paint
   - b) PDF
   - c) MS Word
   - d) Scratch

2. பல மடக்கோப்புகள் சேமிக்கப்படும் இடம்

   - அ) மடக்கோப்புக்கோப்பு
   - ஆ) பட்டி
   - இ) Paint
   - ஈ) ஸ்கேனர்

3. நிரல் (script) உருவாக்கப் பயன்படுவது எது?

   - அ) Script area
   - ஆ) Block palette
   - இ) Stage
   - ஈ) Sprite

4. நிரல்தாக்கத்திற்குப் பயன்படுவது எது?

   - அ) Inkscape
   - ஆ) Script editor
   - இ) Stage
   - ஈ) Sprite

5. தொகுக்குகளை (Block) உருவாக்கப் பயன்படுவதெது?
   - அ) Block palette
   - ஆ) Block menu
   - இ) Script area
   - ஈ) Sprite

### II. பொருத்துக

| A                     | B              |
| --------------------- | -------------- |
| 1. நிரல்தாக்கப்பகுதி  | Script Area    |
| 2. மடக்கோப்புக்கோப்பு | Folder         |
| 3. ஸ்கிராட்ச்         | Scratch        |
| 4. ஆடைத்திருத்தி      | Costume editor |
| 5. நோட்பேடு           | Notepad        |

### III. சுருக்கமான விடையளி

1. ஸ்கிராட்ச் (SCRATCH) என்றால் என்ன?
2. திருத்தி (EDITOR) குறித்தும் அதன் பகுதிகள் குறித்தும் எழுதுக?
3. மேடை (STAGE) என்றால் என்ன?
4. ஸ்பிரைட் (SPRITE) என்றால் என்ன?

## சுருக்கம்

| கருத்து                     | விளக்கம்                        |
| --------------------------- | ------------------------------- |
| மடக்கோப்பு (File)           | தரவுகளைச் சேமிக்கும் அலகு       |
| மடக்கோப்புக்கோப்பு (Folder) | பல மடக்கோப்புகளைக் கொண்ட கோப்பு |
| ஸ்கிராட்ச் (Scratch)        | காட்சி நிரல்தேனிழி              |
| ஸ்டேஜ் (Stage)              | காட்சி மேடை                     |
| ஸ்பிரைட் (Sprite)           | கதாபாத்திரங்கள்                 |
| நிரல் (Script)              | செயல்பாட்டு வழிமுறைகள்          |
| தொகுக்கு (Block)            | நிரல் கட்டுமான அலகுகள்          |
