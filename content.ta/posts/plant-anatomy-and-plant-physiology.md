## பயிற்சி

### I. சரியான விடையைத் தேர்வு செய்க

1. காஸ்மபரியன் பட்டைகள் மூலரின் ________ பகுதியில்
   - அ. புறணி
   - ஆ. பித்
   - இ. தபரிணைக்கிள்
   - ஈ. அகத்தால்

2. உள்மாக்கிய நெலம் என்பது என்ன சிறப்புப் பண்பாகும்?
   - அ. மூலர்
   - ஆ. ண்டு
   - இ. இலைகள்
   - ஈ. ஏலர்கள்

3. நெலமும் பும்ளதாயமும் ஒரு ஆரத்தில் அருகருக அமைந்து காணப்படுவது ________ எனப்படும்.
   - அ. ஆரப்பதாக்கு அமைப்பு
   - ஆ. நெலம் சூழ் வாஸ்குலார் கறை
   - இ. ஒன்றிணைந்தவை
   - ஈ. இவற்றில் எதுவும் இல்லை

4. காற்றில்லாச் சுவாசத்தின் மூலம் உருவாவது
   - அ. காரமபதாணைட்மரட்
   - ஆ. எதிலும் ஆல்கைதால்
   - இ. அசிட்டணடல் ம்கா
   - ஈ. ணபருமவட்

5. கிரப் சுழற்சி இங்கு நுண்டுளை பறுகிறது
   - அ. பசுங்கணிகம்
   - ஆ. நைட்மடதா காண்ட்ரியாவின் உட்கூழ் மேம் மேட்ரிக்ஸ்
   - இ. புறத்தால் துண்ள்
   - ஈ. நைட்மடதா காண்ட்ரியாவின் உட்புறச் சவ்வு

6. ஒளிச்சேர்க்கையின் பதாது எந் நிலையில் ஆக்ஸிஜன் உற்பத்தியாகிறது?
   - அ. ATP யதானைது ADP யதாக மேற்றும் பதாது
   - ஆ. CO₂ நிலை நிறுத்தப்படும் பதாது
   - இ. நீர் மூலக்கூறுகள் பிளக்கப்படும் பதாது
   - ஈ. இவை

### II. காலி இடங்களை நிரப்புக

1. மூலரில் புறணியின் உட்புற அடுக்கு ____________ ஆகும்
2. நெலமும் பும்ளதாயமும் தவ மறு ஆரங்களில் காணப்படும் வாஸ்குலார் கறை ____________ அமைவாகும்
3. கிளைக் காலிசிஸ் நுண்டுளை பறும் இடம் ____________
4. ஒளிச்சேர்க்கையின் பதாது வெளிப்படும் ஆக்ஸிஜன் ____________லிருந்து கிடைக்கிறது.
5. செல்லின ATP உற்பத்தித் தாழிறைதாணல் ____________

### III. சரியா? தவறா? (தவறு எனில் கூற்றினைத் திருத்துக)

1. தாவரங்களில் நீணர் கடத்துவதில் ஈடுபடும் திசு பும்ளதாயம்.
2. தாவரத்தின் வெளிப்புறத்தில் காணப்படும் தேழுகுப்படலம் கியூடிக்கிள்
3. ஒருவினையினலத் தாவர ண்டில் நெலத்திறகும் பும்ளதாயத்திறகும் இடையில் ம்கம்பியம் காணப்படுகிறது.
4. இருவினையினலத் தாவர மூலரில் மேற்புறத் தோலுக்குக் கீழாக பாலிமெட்பதாரணகேதா உள்ளது.
5. இலையிணைத திசு பசுங்கணிகங்களைப் பற்றறுள்ளது.
6. காற்றுச் சுவாசத்தின் விட காற்றில்லாச் சுவாசம் அதிக ATP மூலக்கூறுகளை உற்பத்தி

### IV. பொருத்துக

| A | B |
|---|-----|
| 1. வாஸ்குலார் கறை | - உயிரு கடத்துதல் |
| 2. ம்கம்பியம் | - தபரணிகள் |
| 3. நெலம் சூழ் வாஸ்குலார் கறை | - இரண்டாம் நிழல வளர்ச்சி |
| 4. நெலம் | - |
| 5. பும்ளதாயம் | - நீணர் கடத்துதல் |

### V. விடையளி

1. எனனை?
2. ஒளிச்சேர்க்கைக்கும் பவையானை கார்பன் எதிலிருந்து பெறப்படுகிறது?
3. காற்றுச் சுவாசத்திறகும் காற்றில்லாச் சுவாசத்திறகும் தபாதுவதானை நிகழ்ச்சி எது?
4. காரமபதாணைட்மரட்டதானைது ஆக்ஸிகரை மேணடந்து ஆல்கைதாலதாக வளிமயறும் நிகழ்வின் பெயர் என்ன?

### VI. விரிவாக எழுதுக

1. வாஸ்குலார் கறையின் அமைப்புப்பற்றி எழுதுக.
2. இலையிணைத திசு (மீமெதாபில்) பற்றிக் குறிப்பு எழுதுக.
3. ஒரு ஆக்ஸிமஸைதாமின் படம் வரைந்து பாகங்களைக் குறி.
4. ஏலரும் தாவரங்களில் காணப்படும் மூன்று வகையான திசுத்தொகுப்புகளைக் குறிப்பிடுக.
5. ஒளிச்சேர்க்கை என்றால் என்ன? இது செல்லில் எங்கு நுண்டுளை பறுகிறது?
6. சுவாச ஈவு என்றால் என்ன?
7. ஒளிச்சேர்க்கையின் பதாது இருள்வினைக்கு முன்பு ஏன் ஒளிவினை நுண்டுளை பைமவண்டும்?
8. ஒளிச்சேர்க்கையின் ஒட்டுத்தேதாத் தேன்பதாட்டும் எழுதுக.

### VII. விரிவான விடையளி

1. அ. ஒருவினையினலத் தாவர மூலர் பெற்றும் இருவினையினலத் தாவர மூலர்
   ஆ. காற்றுள்ள சுவாசம் பெற்றும் காற்றில்லாச் சுவாசம்

2. காற்றுச் சுவாசிகள் தெல் சுவாசத்தின் பதாது எவ்வாறு குளுக்கோஸிலிருந்து ஆற்றலைப் பற்றுகின்றன? அற்கான மூன்று படிநிலைகளை எழுதி விவரிக்கவும்.

3. ஒளிச்சேர்க்கையின் ஒளி எதார்ந்த் தெயல் எவ்வாறு ஒளிச் எதாரதாத் தெயலிலிருந்து மறுபடுகிறது. இந் நிகழ்ச்சியின் ஈடுபடும் மூலப் தபாருள்கள் யதாணவ? இறுதிப் தபாருட்கள் யதாணவ? இவ்விரு நிகழ்ச்சிகளும் பசுங்கணிகத்தில் எங்கு நுண்டுளை பறுகின்றன?

### VIII. திறன் வளர்ச்சி

1. ஒளிச்சேர்க்கை ஒரு உயிரமவதி இயல் நிகழ்ச்சியாகும்.
   - அ. ஒளிவினை பெற்றும் இருள்வினையின் பதாது உருவாகும் வினை விளை பதாருட்கள் யதாணவ?
   - ஆ. ஒளிச்சேர்க்கையின் உயிரமவதி வினையில் ஈடுபடும் சில வினைபடு தபாருட்கள் இந் நிகழ்ச்சியின் சுழற்சியில் மீண்டும் மீண்டும் ஈடுபடுகின்றன அந் வினைபடு தபாருட்களைக் குறிப்பிடுக.

2. பசுங்கைகித்தின் எந் பகுதியில் ஒளிச் எதார்ந்த் தெயல் பெற்றும் கால்வின சுழற்சி நுண்டுளை பறுகின்றன?
