# அலகு 23: காட்சித் தொடர்பு

தற்போதுவரை கணினி என்றால் மடிக்கணினி, திரை, விசைப்பலகை, சுட்டி, மெய்நிகர் மற்றும் போன்றவை நினைவுக்கு வரும். கணினி, கணினியின் பாகங்கள் மற்றும் கணினியைக் குறித்த அறிமுகத்தை ஏழாம் வகுப்பில் அறிந்திருப்போம். அவற்றைத் தவிர கணினியை இயக்குவதில் சில மென்பொருட்களும் தேனிப்பொருட்களும் முக்கியப் பங்காற்றுகின்றன. நம் மனவளுக்கு ஏற்றவாறு கணினியை எவ்வாறு செயல்படுத்துவது என்பதையும் இனி அறிந்திடுவோம்!

![](img1.png)

கணினியை நாம் நாடுவதற்கான காரணம் அனைத்து வகை மெமிப்புத்திறனுடனாகும். கணினியில் எவ்வாறு நம் கவலைகளைச் சேமித்துணவப்பது? பல கோப்புகள் உள்ளடங்கிய மடக்கோப்பு அல்லது கோப்பகம் என்பதன் மூலம் சேமித்துணவக் கலாம். இதன்மூலம் கணினியில் மடக்கோப்பும் (Files) மடக்கோப்புக்கோப்பும் (Folder) எனப்படும் தெரிந்து கொள்ளலாம்.

## மடக்கோப்புக்கோப்பு (Folder)

**மடக்கோப்புக்கோப்பு என்றால் என்ன?**

மடக்கோப்புக்கோப்பு என்பது பல மடக்கோப்புகளை உள்ளடக்கிய பட்டைக் கோப்பு ஆகும். இவற்றைத் தம்மனவக் கற்பனையால் உருவாக்கிக்கொள்ளமுடியும்.

&gt; **எடுத்துக்காட்டு:** நம் வீட்டில் உள்ள புத்தக அலமாரிகளில் உள்ள ஒரு புத்தகம் என்பது மடக்கோப்பு என்றும், புத்தகம் முழுவதையும் தாண்டி அலமாரியை மடக்கோப்புக்கோப்பு என்றும் எளிதில் கூறிவிடலாம்.

![](folder.png)

### புதிய மடக்கோப்புகளை உருவாக்குவது எப்படி?

தற்போதைக்கு சுட்டியின் வலப்புறப் பொத்தானை அழுத்தியதும் கணினித்திரையில் **New** எனத் தோன்றும். அதில் **Folder** எனச் சொடுக்கினால் புதிய மடக்கோப்புக்கோப்பு நம் பயன்பாட்டிற்குத் தயாராகிவிடும்.

![](img2.png)

அதிகமானவர்களால் பயன்படுத்தப்படும் இயக்கமேனிப்பொருளான விண்டோஸ் மற்றும் லினக்ஸில் குறிப்புகளைச் சேமித்தல், படங்களை வரைதல், அனிமேஷன் படங்களைத் தயாரித்தல் மற்றும் பலதெயல்களில் தனித்தனியாக நம் கையால் செய்யலாம்.

எப்படி முன்பின் தெரியாத ஊர்களுக்குச் செல்ல ஆங்கிலம் மொழிக்கோட்டிப்பலகைகள் இருக்கின்றனவோ அம்மாதிரி கணினியை இயக்கத் தொடங்கியவுடன் இடப்புறத்தின் கீழ் உள்ள **START** எனச் சொடுக்கிய உடன் கணினியில் உள்ள அனைத்து நிரல்களின் பட்டியல் திரையில் காட்டப்படும். அதில் நேர்குதம் மனவயானைத் தேர்ந்தெடுத்துக்கொள்வதன் மூலம் அந்நிரல்களில் மனவக் கற்பனை மடக்கோப்புகளை உருவாக்கலாம்.

## மடக்கோப்புகளை உருவாக்குவது எப்படி?

தற்போதைக்கு விண்டோஸ் இயங்குடைய உள்ள கணினிகளில் நம் குறிப்புகளைச் சேமித்துணவக்கும் நோட்பேட் (Notepad) தெயலியையும், படங்களை வரையத் பெயிண்ட் (Paint) என்னும் தெயலியையும் தேர்ந்தெடுக்கலாம்.


அதன்படி **NotePad** இல் மனவயான குறிப்புகளைத் தட்டச்சுச் செய்வதன் மூலம் உருவாக்கப்பட்ட மடக்கோப்புகளை மடக்கோப்புக்கோப்புகளில் சேமித்துணவுதலாம். இதுமாதிரி **Paint** என்னும் தெயலியில் ஒரு படத்தை உருவாக்கவோ அல்லது உருவாக்கிய படங்களைத் திருத்தம்செய்யவோ இயலும். இப்படங்களைத் தாண்டி படக்கோப்பினை உருவாக்குவது பற்றியும், அனிமேஷன் படங்களையும் வரைகலைப் படங்களையும் எளிதாக உருவாக்குவது பற்றியும் அதுதொடர்பான தெயலிகளின் பயன்பாடுகளைப் பற்றியும் இனிக் காண்போம்.

## ஸ்கிராட்ச் (SCRATCH) மென்பொருள்

### அறிமுகம்

விளையாட்டுகளையும் எளிதில் உருவாக்கப் பயன்படும் ஒரு தேனிப்பொருள் **ஸ்கிராட்ச் (SCRATCH)**. இது ஒரு காட்சி நிரல்தேனிழி (Visual Programming Language). எம்ஐடி (Massachusetts Institute of Technology - MIT) என்னும் பல்கலைக்கழக நுட்ப ஆய்வகம் இந்நிரலை எளிதாகவும் மடிக்கையாகவும் கற்கும் வகையில் வடிவமைத்துள்ளது.

### முக்கிய பகுதிகள்

ஸ்கிராட்ச் தேனிப்பொருள் மூன்று பிரிவுகளை உள்ளடக்கிய ஒரு தேனிழம்பு புலப்படும்:

| பகுதி                                   | விளக்கம்                                         |
| --------------------------------------- | ------------------------------------------------ |
| **ஸ்டேஜ் (Stage)**                      | காட்சி மேடை - நிறைந்த பின்னணி தவழ்வதாக இருக்கும் |
| **ஸ்பிரைட் (Sprite)**                   | கணினி கதாபாத்திரங்கள் (Characters)               |
| **ஸ்கிரிப்ட் எடிட்டர் (Script Editor)** | நிரல் திருத்தி                                   |

### ஸ்கிராட்ச் இடைமுகம்

இடப்புறம் மேல்பகுதியில் **ஸ்டேஜ்** பிரிவும், இடப்புறம் கீழ்ப்பகுதியில் **ஸ்பிரைட் பட்டியல்** பிரிவும், வலப்புறத்தில் **ஸ்கிரிப்ட் எடிட்டர்** பிரிவும் இருக்கும்.

**மூன்று தாவல்கள்:**

- Script (நிரல்)
- Costume (ஆடை)
- Sound (ஒலி)

## ஸ்கிராட்ச் பயன்பாடு

### அடிப்படை நிரல் உருவாக்கம்

**தேவை:** பச்சைக்கொடி சொடுக்கும்போது ஸ்பிரைட் நகர வேண்டும்

**படிநிலைகள்:**

1. **File → New** ம் தேர்ந்தெடுக்க. புதிய தெயல்திட்டம் (Project) உருவாகும்

2. **Script Menu** வில் **Event** ஐத் தேர்ந்தெடுக்க. இப்போது சில தொகுகள் தோன்றும். **"When green flag clicked"** என்ற block ஐ drag செய்து Script area வில் இணக்கவும்.

   ![](img4.png)

3. **Script → Motion menu** ணவதம் தேர்ந்தெடுத்து **goto x:0 y:0** என்ற block ஐ drag செய்து ஏற்கனவே உள்ள "When green flag clicked" blockன் அடியில் பொருத்தவும்.

   ![](img5.png)

4. **"move 10 steps"** என்ற block ஐ ஏற்கனவே உள்ள இரண்டு block களுக்கும் அடியில் பொருத்தவும். மேலும் "move 10 steps" என்பதில் உள்ள 10 ஐ 100 ஆக மாற்றவும்.

   ![](img6.png)

5. இந்நிரலை இயக்க வலப்புறம் மேல்பகுதியில் உள்ள பச்சைக்கொடி சொடுக்கவும்.

## ஒலியைச் சேர்த்தல்

1. **Sound tab** ஐத் தேர்ந்தெடுக்க. இப்போது தெரியும் Sprite list இல் 'Sprite1' என்பதைத் தேர்ந்தெடுக்கவும்.

2. **'மியாவ்'** என்ற ஒலியைக் கேட்கும். வேறு ஒலி வேண்டுமெனில் speaker icon ஐத் தேர்ந்தெடுத்து Sound library இலிருந்து தேவையான ஒலிகளைப் பெறலாம்.

3. **Script tab → Sound → Play sound block** ஐத் தெரிவுசெய்க. அதனுடன் **"When space key pressed"** block ஐச் சேர்க்கவும்.

4. நிரலை இயக்கவும்.

## எடுத்துக்காட்டு: "Hello" என்று சொல்லும் நிரல்

"Hello" என்று தெளிவான ஒலியுடன் பதிவுசெய்யும் நிரல்:

![](<img7(0).png>)

1. **Script tab** இல் **Events** option ஐத் தேர்ந்தெடுக்க.

   ![](img7.png)

2. **"When green flag clicked"** என்ற block ஐ script area விற்குள் இழுத்து இணக்கவும்.

   ![](img8.png)

3. **Script tab** இல் **looks** option ஐத் தேர்ந்தெடுக்க. **Say ""** என்ற block ஐ Script area விற்குள் இழுத்து இணக்கவும்.

   ![](img9.png)

4. **Say** tab னுள் **"Hello"** என்று வார்த்தையிடவும்.

   ![](img10.png)

5. **Script tab → Sounds** option ஐத் தேர்ந்தெடுக்க. **"Play sound"** block ஐ இழுத்து Script Area வினுள்விடவும். Audio file இலிருந்து **"Hello"** sound ஐத் தேர்ந்தெடுக்கவும்.

   ![](img11.png)

6. **File menu → Save** ம் தேர்ந்தெடுக்க. இப்போது உங்கள் Project சேமிக்கப்படுகிறது.

7. நிரலை இயக்க மேல் வலதுபுறத்தில் உள்ள பச்சைக்கொடி சொடுக்கவும்.

   ![](img12.png)
