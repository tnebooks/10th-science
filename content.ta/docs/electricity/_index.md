# அலகு 4 :மின்னோட்ட வியல்

## கற்றல் விளைவுகள்
* மின்சுற்றுக்களை உருவாக்குதல்.
* மின்னழுத்தம் மற்றும் மின்னழுத்த வேறுபாட்டை வேறுபடுத்துதல்.
* மின்னோட்டம் மற்றும் மின்கடத்துதிறனைப் பற்றி உயர்ந்து கொள்ளுதல்.
* மின்னடையாக்கிகளின் தொடர்ஓட்டம் மற்றும் பக்க இணைப்பு மற்றும் இதன் தாக்கங்களுக்குத் தீர்வு காணுதல்.
* மினமைதாட்டத்தின் வெப்பவிளைவுப் பற்றி அறிந்து கொள்ளுதல் மற்றும் அன்றாட வாழ்வில் இணை பயன்படுத்துதல்.
* மின்திறன் மற்றும் மின்னறைல் வணரயறுதல் மற்றும் வீட்டுக்கான மின்சுற்றுகள் பற்றிய விளக்கங்களைக் கேளிதல்.
* LED விளக்கு மற்றும் LED திரைக்காட்சிகளின் நவீனப் பயன்பாடுகள் பற்றி அறிதல்.

## அறிமுகம்
ஏற்கனவே படித்திருப்பீர்கள். ஒரு கடத்தி வழியாக மின்னூட்டங்களின் இயக்கத்தைப் பற்றிக் கூறுவது மினமைதாட்டம் ஆகும். மினமைதாட்டம் என்பது ஒரு வண்கயதானை ஆறைல். மினமைதாட்டத்தின் பயன்கள் நம் அன்றாட வாழ்வில் இன்றியமையாதாகவும், விரக்கமுடியாதாகவும் இருக்கின்றன. வீடுகள், கல்வி நிறுவனங்கள், சேருதல் வேணைகள் மற்றும் தொழிற்சாலைகளில் மிகப் பொதுவான மினமைதாட்டத்தின் பயன்கள் பற்றி நீங்கள் அறிந்திருப்பீர்கள். இப்பாடத்தில் மினமைதாட்டம் பற்றியும் மினமைதாட்டத்தினால் ஏற்படும் விளைவுகள் பற்றியும் பாரக் கற்றுக் கொள்வோம்.

### 4.1 மின்னோட்டம்
மின்னூட்டங்களின் (எலக்ட்ரான்களின்) இயக்கமே மினமைதாட்டம் ஆகும். ஒரு கால்வாய் வழியாக ஓடும் நீரம் போல அல்லது உயர அழுத்தப் பகுதியிலிருந்து குறைந்த அழுத்தப் பகுதியை நோக்கி வீசும் காற்றினைப் போல, எலக்ட்ரான்கள் கடத்தியின் வழியாகப் பாய்ந்து செல்வது மினமைதாட்டம் எனகிறது.

**மின்னோட்டத்தின் திசை:** எலக்ட்ரான்களின் இயக்கத்திற்கு எதிர்திசையில், உயர மின்னழுத்தத்தில் இருக்கும் நேர மின்முனையிலிருந்து குறைந்த மின்னழுத்தத்தில் இருக்கும் எதிர்முனை ஓட்டம்

![எலக்ட்ரான் பாயும் திசை](image)

**படம் 4.1** எலக்ட்ரான் பாயும் திசை

#### 4.1.1 மின்னோட்டத்தின் அளவு
குறிக்கப்படுகிறது. கடத்தி ஒன்றின் ஒரு பகுதியின் வழியாக ஒரு மின்னூட்டங்கள் பாயும் வீச்சு மினமைதாட்டம் என வணரயறுக்கப்படுகிறது. அதாவது ஒரு அலகு நேரத்தில் கடத்தியின் ஒரு குறுக்குத்தட்டுப் பகுதியைக் கடந்து செல்லும் மின்னூட்டங்களின் அளவு மினமைதாட்டமே ஆகும். ஒரு கடத்தியின் குறுக்குத்தட்டுப் பகுதி வழியாக Q அளவு மின்னூட்டம் 't' காலத்தில் கடந்து சென்றால் அதில் பாயும் மினமைதாட்டமே ஆனது

{{< katex display=true >}}
(I = \frac{Q}{t}) {{< /katex >}}

#### 4.1.2 மின்னோட்டத்தின் அலகு
கூலும் மின்னூட்டம் ஒரு வினாடி நேரத்தில் கடத்தியின் எத்தாவது ஒரு குறுக்குத்தட்டுப் பகுதி வழியாகக் கடந்து செல்லும் போது அக் கடத்தியில் பாயும் மினமைதாட்டம் ஒரு ஆம்பியர் என வணரயணை செய்யப்படுகிறது. எனவே

{{< katex display=true >}}
(1 \text{ ஆம்பியர்} = \frac{1 \text{ கூலும்}}{1 \text{ வினாடி}}) {{< /katex >}}

**தீர்க்கப்பட்ட கணக்கு 1**
மின்விளக்கின் வழியாகப் பாய்கிறது எனில் அவ்வழியாகப் பாயும் மினமைதாட்டத்தின் அளவு என்ன?

**தரவுகள்:**
மின்னூட்டம் Q = 12 கூலும், காலம் t = 5 வினாடி.

எனவே, மினமைதாட்டம் I = Q/t = 12/5 = 2.4 A

### 4.2 மின்சுற்று
வழியாக அனுமதிக்கும் பல மின்கூறுகளின் வணலயணைப்புத் கொண்டு உருவாக்கப்பட்ட ஒரு மூடிய சுற்று அல்லது பாதையாகும். மின்சாரங்களைக் கண்காணிக்கும் மின்னூட்டத்தின் மூலேதானை மின்கலத்தையும் இணைக்கும் பாதைகளைக் கம்பிகள் பயன்படுத்தப்படுகின்றன. மின்கலம், மின்விளக்கு, தொழிற்சாலை ஆகியவைக் கொண்டு உருவாக்கப்பட்ட ஒரு எளிய மின்சுற்று படம் 4.3ல் காட்டப்பட்டுள்ளது.

![எளிய மின்சுற்றுப் படம்](image)

**படம் 4.2** எளிய மின்சுற்றுப் படம்

இம் மின்சுற்றில் தொழிற்சாலை மூடியிருக்கும் போது மின்விளக்கு ஒளிருகிறது. தொழிற்சாலை திறந்திருக்கும் போது மின்விளக்கு ஒளிராது. எனவே, மினமைதாட்டம் பாய்வதற்கு அச்சுற்றுப் பாதை மூடப்பட வேண்டும். மினமைதாட்டம் பாய்வதற்கு வேண்டியதானை மின்னழுத்த வேறுபாட்டினை மின்கலம் வழங்குகிறது.

**அட்டவணை 4.1 மின்கூறுகள் மற்றும் அவற்றின் குறியீடுகள்**

| மின்கூறு | மின்கூறின் பயன்பாடு | குறியீடு |
|---------|------------------|---------|
| மின்னடையாக்கி | மின் சுற்றில் பாயும் மினமைதாட்டத்தின் அளவை நிர்ணயம் செய்ய பயன்படுகிறது. | ⎓ |
| மின்னோட்ட மீட்டர் | மினமைதாட்டத்தின் அளவை அளவிடுக்கப் பயன்படுகிறது. | அம்மீட்டர் |
| மின்னழுத்த மீட்டர் | மின்னழுத்த வேறுபாட்டின் அளவிட | வோல்ட் மீட்டர் |
| கால்வனோ மீட்டர் | மினமைதாட்டம் மற்றும் அவ் திசையைக் கண்டறிய | கல்வனோ மீட்டர் |
| மின்னணு கருவிகளில் பயன்படுகிறது | டையோடு |
| ஒளிமின் டையோடு (LED) | ஏழு துண்டு காட்சிப் பலகையில் பயன்படுகிறது |
| திற இணைப்பு | மின் சாரங்களைப் பாதுகாக்கப் பயன்படுகிறது. |
| மின்னழுத்தத்தின் அளவிட குறிப்பு புள்ளியாகச் செயல்படுகிறது. |

எலக்ட்ரான்கள் மின்கலத்தின் எதிர்மினமுனையிலிருந்து நேர்மினமுனைக்குச் செல்கின்றன. மின்னூட்டத்தின் திசையில் இருக்கும். அல்லது எதிர்மினமைதாட்டம் செல்லும் திசைக்கு எதிர்திசையில் அணைந்திருக்கும் எனவும் கூறலாம். எனவே, மினமைதாட்டத்தின் திசைதானைது ஒரு மின்சுற்றில் நேர்மினமுனையிலிருந்து எதிர்மினமுனையை நோக்கி இருக்கும்.

### 4.3 மின்னழுத்தம் மற்றும் மின்னழுத்த வேறுபாடு
நீர்மதாட்டம் மற்றும் காற்றமைதாட்டம் பற்றி ஏற்கனவே கீழ்வகுப்புகளில் படித்திருப்பீர்கள். ஒரு திண்மப் பொருளில் இரண்டு புள்ளிகளுக்கிடையே தவப்ப நிலை வேறுபாடு இருந்தாலே அவ்வழியாகத் தவப்பம் பாயும் என்பது உங்களுக்குத் தெரியும். இம்மாதிரி ஒரு கடத்தியில் இரு புள்ளிகளுக்கிடையே மின்னழுத்த வேறுபாடு இருந்தாலே அக் கடத்தியில் மின்னூட்டம் பாயும். ஒரு கடத்தியில் மின்னூட்டமே ஆனது உயர் மின்னழுத்தப் புள்ளியிலிருந்து குறைந்த மின்னழுத்தப் புள்ளிக்குப் பாயும்.

#### 4.3.1 மின்னழுத்தம்
நேர்மின்னூட்டத்தின் முடிவில்லாதால் இருந்து மினவிசைக்கு எதிராக அப்புள்ளிக்குக் கொண்டுவரப்படும் முயற்சி என வணரயறுக்கப்படுகிறது.

#### 4.3.2 மின்னழுத்த வேறுபாடு
மின்னழுத்த வேறுபாடு என்பது ஒரு புள்ளியிலிருந்து வேறொரு புள்ளிக்கு ஒரு அலகு நேர்மின்னூட்டத்தின்

மினவிலக்கு விசைக்கு எதிராக நகர்த்தப்படும் முயற்சி என வணரயறுக்கப்படுகிறது.

![மின்னழுத்த வேறுபாடு](image)

**படம் 4.3** மின்னழுத்த வேறுபாடு

A புள்ளியிலிருந்து B என்ற புள்ளிக்கு நகர்த்தி உள்ளதாகக் கருதுவோம். இம் மின்னூட்டத்தின் A யிலிருந்து B க்கு நகர்த்துவதற்கு செய்யப்பட்ட முயற்சி W எனக் கொள்வோம். A மற்றும் B க்கு இடையே உள்ள மின்னழுத்த வேறுபாடு கீழ்க்கண்ட வகையிலிருந்து பெறப்படுகிறது.

{{< katex display=true >}}
(\text{மின்னழுத்தம்} = \frac{\text{முயற்சி}}{\text{மின்னூட்டம்}}) {{< /katex >}}

இரண்டு புள்ளிகளுக்கு இடையே உள்ள மின்னழுத்தங்களின் வேறுபாட்டினையும் மின்னழுத்த வேறுபாடு எனக் கூறலாம். V_A மற்றும் V_B என்பது புள்ளி A மற்றும் B இல் உள்ள மின்னழுத்தங்கள் எனக் கொண்டால் இவ்விரண்டு புள்ளிகளுக்கிடையேயுள்ள மின்னழுத்த வேறுபாடு

{{< katex display=true >}}
(V = V_A - V_B (V_A > V_B)) {{< /katex >}}
{{< katex display=true >}}
(V = V_B - V_A (V_B > V_A)) {{< /katex >}}

#### 4.3.3 மின்னழுத்த வேறுபாட்டின் அலகு
மின்னழுத்த வேறுபாட்டின் அலகு வோல்ட் (V) ஆகும்.

ஒரு புள்ளியிலிருந்து வேறொரு புள்ளிக்கு மினவிசைக்கு எதிராக எடுத்துச் செல்லப்படும் முயற்சியின் அளவு ஒரு ஜூல் எனில் அப்புள்ளிகளுக்கிடையே உள்ள மின்னழுத்த வேறுபாடு ஒரு வோல்ட் ஆகும்.

{{< katex display=true >}}
(1 \text{ வோல்ட்} = \frac{1 \text{ ஜூல்}}{1 \text{ கூலும்}}) {{< /katex >}}

**தீர்க்கப்பட்ட கணக்கு 2**
இரண்டு புள்ளிகளுக்கிடையே நகர்த்தப்படும் முயற்சி 100 J எனில் அப்புள்ளிகளுக்கிடையே உள்ள மின்னழுத்த வேறுபாடு என்ன?

**தீர்வு:**
{{< katex display=true >}}
(V = \frac{W}{Q} = \frac{100}{10} = 10 \text{ V}) {{< /katex >}}

### 4.4 ஓமின் விதி
மினமைதாட்டம் மற்றும் மின்னழுத்த வேறுபாடு ஆகியவற்றிற்கிடையேயான தொடர்பை நிறுவினைதார். இது ஓமின் விதி எனப்படும்.

![ஓமின் விதி சோதனை](image)

**படம் 4.4** ஓமின் விதி சோதனை

இவ்விதியின்படி வெப்பநிலையில், கடத்தி ஒன்றின் வழியாகப் பாயும் சீரான மினமைதாட்டம் கடத்தியின் முனைகளுக்கிடையே உள்ள மின்னழுத்த வேறுபாட்டிற்கு நேர்க்கவில் அணையும்.

{{< katex display=true >}}(I \propto V \text{ எனவே, } \frac{I}{V} = \text{ மாறிலி}) {{< /katex >}}இம் மாறிலி மதிப்பு {{< katex display=true >}}(\frac{1}{R}) {{< /katex >}} ஆகும்.எனவே, {{< katex display=true >}}(I = \left(\frac{1}{R}\right)V{{< / katex >}}( அல்லது ) {{< katex >}}V = IR) {{< /katex >}}

இங்கு R என்பது மின்னடையாகும். ஒரு குறிப்பிட்ட பொருளுக்கு (எ.கா. நிக்கல்) குறிப்பிட்ட வெப்பநிலையில் மின்தடை ஒரு மாறிலி ஆகும். மின்னழுத்த வேறுபாடு V யும் மினமைதாட்டம் I யும் ஒன்றுக்கொன்றும் நேர்க்கவில் அணைவதால் V மற்றும் I இடையேயான தொடர்பு ஒரு நேர்க்கோடு ஆகும். இது படம் 4.5ல் காட்டியுள்ளது.

![மினமைதாட்டம்-மின்னழுத்தம் வரைபடம்](image)

**படம் 4.5** மினமைதாட்டம்-மின்னழுத்தம் வரைபடம்

அணை ஒரு மின்கலம், தொழிற்சாலை மற்றும் மின்னடை மாற்றி ஆகியவற்றுடன் தொடராக இணைக்கவும். தொழிற்சாலை மூடிய நிலையில் மின்னடை மாற்றியில் மாற்றம் செய்து பல்வேறு மின்னழுத்தங்களுக்கு மினமைதாட்டத்தைக் கைக்கிடுங்கள். உங்களுக்குக் கிடைத்த V/I ன் மதிப்பு மாறிலியாக இருப்பதைக் கவனியுங்கள். இம்முறையினை நிக்கலுக்குப் பதிலாகத் தாமிரக் கம்பியினைப் பயன்படுத்திச் செய்து பாருங்கள். இங்கும் V/I ன் மதிப்பு மாறிலியாக இருந்தாலும், ஒரே மின்னழுத்த வேறுபாட்டுக்கு மினமைதாட்டத்தின் மதிப்பு மாறுபடுவதைக் கவனியுங்கள். இதும்போல் தாமிரக் கம்பிக்குப் பதிலாக அலுமினியக் கம்பியைப் பயன்படுத்தும் போதும் ஒரே மின்னழுத்த வேறுபாட்டுக்கு மினமைதாட்டத்தின் மதிப்பு மாறுபடுவதைக் கவனியுங்கள்.

ஒரே மின்னழுத்தத்திற்கு வேறுபட்ட பொருட்களுக்குத் தவறான மினமைதாட்டமேதிப்புகள் கிடைத்திருப்பது, தவறான பொருட்களுக்கு மின்னடை மதிப்புகள் தவறாக இருக்கும் என்பதைக் காட்டுகிறது.

ஒரு பொருளின் வழியாக மின்னூட்டங்கள் பாய்ந்து செல்வது அல்லது மினமைதாட்டம் பாய்வது எதிர்க்கும் பண்பு அப்பொருளின் **மின்தடை** எனப்படும்.

ஒரு பொருளின் மின்னடை என்பது ஒரு பொருளின் வழியாக மின்னூட்டம் பாய்வது (அதாவது மினமைதாட்டம் செல்வது) எதிர்க்கும் பண்பாகும். இது தவறான பொருட்களுக்குத் தவறாக இருக்கும். ஓமின் விதியிலிருந்து

{{< katex display=true >}}(\frac{V}{I} = R \text{ என எழலாம்.}) {{< /katex >}}

மின்னழுத்த வேறுபாட்டிற்கும் அதன் வழியாகச் செல்லும் மின்னோட்டத்திற்கும் இடையேயுள்ள கடத்தியின் மின்தடை என வணரயறுக்கப்படுகிறது.

#### 4.5.1 மின்னடையின் அலகு
Ω என்னும் குறியீட்டால் குறிக்கப்படுகிறது.

மின்னழுத்த வேறுபாடு ஒரு வோல்ட்டாக இருக்கும் போது கடத்தியில் செல்லும் மினமைதாட்டம் ஒரு ஆம்பியர் எனில் அம் மின்னடை ஒரு ஓம் ஆகும்.

{{< katex display=true >}}
1 \text{ ஓம் } = 1 \text{ வோல்ட் } / 1 \text{ ஆம்பியர் }
{{< /katex >}}

**தீர்க்கப்பட்ட கணக்கு 3**
30 V மின்னழுத்த வேறுபாடு கொண்ட கடத்தியின் முனைகளுக்கு இடையே 2 A மினமைதாட்டம் பாய்கிறது. அதன் மின்னடையைக் காண்க.

**தீர்வு:**
மின்னழுத்த வேறுபாடு V = 30 V, மினமைதாட்டம் I = 2 A

ஓமின் விதியின்படி R = V/I

எனவே, R = 30/2 = 15 Ω

### 4.6 மின்கடத்து எண்

#### 4.6.1 மின்னடை எண்
நீளத்திற்கு (L) நேர்க்கவிலும், குறுக்குத்தட்டுப் பரப்பிற்கு (A) எதிர்க்கவிலும் அணையும்.

{{< katex display=true >}}(R \propto L, R \propto \frac{1}{A}, R \propto \frac{L}{A}) {{< /katex >}}எனவே, {{< katex display=true >}}(R = \rho \frac{L}{A}) {{< /katex >}}ρ என்பது மின்னடை எண் எனப்படும்.வகையிலிருந்து, {{< katex display=true >}}(\rho = \frac{RA}{L}) {{< /katex >}}ρ பரப்பும் கொண்ட கடத்தி ஒன்றும் மினமைதாட்டத்திற்கு ஏற்படுத்தும் மின்னடை அக் கடத்திப் பொருளின் மின்னடை எண் என வணரயறுக்கப்படுகிறது. இன் அலகு ஓம் மீட்டர் (Ω m)

வழியாகப் பாயும் மினமைதாட்டத்தினை எதிர்க்கும் திறனைக் குறிக்கும் அளவு ஆகும். ஒரு குறிப்பிட்ட உமைதாகப் பொருளுக்கும் மின்னடை எண் மாறிலி ஆகும்.

#### 4.6.2 கடத்து எண்
பாய்ந்து செல்வது அல்லது மினமைதாட்டம் பாய்வது அனுமதிக்கும் பண்பு அப்பொருளின் மின்கடத்துதிறன் ஆகும்.

என வணரயறுக்கப்படுகிறது. எனவே, ஒரு கடத்தியின் மின்கடத்துதிறன் G என்பது

{{< katex display=true >}}
(\mathbf{G} = \frac{1}{\mathbf{R}}) {{< /katex >}}

குறிப்பிடப்படுகிறது.

**கடத்து எண்** எனப்படும்.

{{< katex display=true >}}
(\sigma = \frac{1}{\rho}) {{< /katex >}}

இன் அலகு ஓம்⁻¹ மீட்டர்⁻¹ அல்லது சீமன்ஸ் மீட்டர்⁻¹ எனவும் குறிப்பிடப்படுகிறது.

மாறிலி ஆகும். மின்கடத்து எண் என்பது ஒரு கடத்தியின் வழியாகப் பாயும் மினமைதாட்டத்தினை அனுமதிக்கும் திறனைக் குறிக்கும் அளவு ஆகும். சில பொருட்கள் மினமைதாட்டத்தை நன்கு கடத்தும். எ.கா. தாமிரம், அலுமினியம் முதலியன. சில பொருட்கள் மின்சாரத்தைக் கடத்தாது (காப்பிகள்) எ.கா. கண்ணாடி, ரப்பர், முதலியன. காப்பிகளைவிட கடத்திகளுக்கு மின்கடத்து எண் அதிகம். ஆனால் மின்னடை எண்ணினைதானைக் காப்பிகளைவிட கடத்திகளுக்குக் குறைவு. தபதாதுவாகப் பயன்படும் சில பொருட்களின் மின்னடை எண் மதிப்பு அட்டவணை 4.2யில் காட்டுக்கப்பட்டுள்ளது.

**அட்டவணை 4.2 மின்னடை எண் மற்றும் மின்கடத்து எண்**

|-------------|--------|------------------|
| கடத்திகள் | தாமிரம் | 1.62 × 10⁻⁸ |
| | நிக்கல் | 6.84 × 10⁻⁸ |
| | குமரதாமியம் | 12.9 × 10⁻⁸ |
| காப்பிகள் | கண்ணாடி | 10¹⁰ முதல் 10¹⁴ |
| | இரப்பர் | 10¹³ முதல் 10¹⁶ |

**தீர்க்கப்பட்ட கணக்கு 4**
10 மீட்டர் நீளமும், 2 × 10⁻⁷ m² குறுக்குத்தட்டுப் பரப்பும் கொண்ட கம்பியின் மின்னடை 2 ஓம் எனில் அவ் (i) மின்னடை எண், (ii) மின்கடத்துதிறன் மற்றும் (iii) மின்கடத்து எண் ஆகியவற்றைக் காண்க.

**தீர்வு:**
நீளம், L = 10 மீ, மின்னடை R = 2 ஓம்
குறுக்குத்தட்டுப் பரப்பு, A = 2 × 10⁻⁷ m²

மின்னடை எண், ρ = RA/L = (2 × 2 × 10⁻⁷)/10 = 4 × 10⁻⁸ Ωm

மின்கடத்துதிறன், G = 1/R = 1/2 = 0.5 mho

மின்கடத்து எண், σ = 1/ρ = 1/(4 × 10⁻⁸) = 0.25 × 10⁸ = 2.5 × 10⁷ S/m

> **உங்களுக்குத் தெரியுமா?**
> நிக்கல்தாம் என்பது மிக உயர்ந்த மின்னடை எண் கொண்ட ஒரு கடத்தியாகும். இன் மதிப்பு 1.5 × 10⁻⁶ Ω m. எனவே இது தவப்பமேற்றும் சாதனங்களில் பயன்படுகிறது.

### 4.7 மின்னடையாக்கிகளின் இணைப்பு
மினமைதாட்டத்தை எவ்வாறு பதாதிக்கிறது என்பதை நீங்கள் இதுவரையில் கற்றுக் கொண்டீர்கள். ஒரு மின்னடையுடைய எளிய மின்சுற்று பற்றியும் அறிந்து கொண்டீர்கள். நேரடி முனையில் சில சிக்கலான மின்சுற்றுக்களை நீங்கள் எதிர்கொள்ள நேரிடும். ஒன்றுக்கும் மேற்பட்ட மின்னடையாக்கிகளின் தொகுப்புகள் மின்சுற்றுக்களுக்கு இணைக்கப்பட்டிருக்கலாம். இணைப்பு மின்னடையாக்கிகளின் அணைப்பு அல்லது மின்னடையின் குழு என அழைக்கலாம். மின்னடைகள் இரண்டு அடிப்படையான முனைகளில் இணைக்கலாம்.

அ) தொடர் இணைப்பில் மின்னடையாக்கிகள்
ஆ) பக்க இணைப்பில் மின்னடையாக்கிகள்

இணைப்பில் இணைக்கப்படும்போது அவற்றின் தொகுபயன் மின்தாணடையைக் கைக்கிடும் முனையைப் பின்வரும் பிரிவுகளில் நீங்கள் காலாம்.

#### 4.7.1 தொடர் இணைப்பு
மின்கூறுகளை ஒன்றன் பின் ஒன்றாக இணைத்து ஒரு மூடிய சுற்றை உருவாக்குவது ஆகும். தொடர் சுற்றில் மினமைதாட்டமே ஆனது ஒரே ஒரு மூடிய சுற்றின் வழியாகப் பாயும். இம் மூடிய சுற்றில் உள்ள ஏதுமொரு புள்ளியில் இணைப்பு ஏற்பட்டதால் மின்சுற்றின் வழியாக மினமைதாட்டம் பாயாது. எனவே சுற்றில் இணைக்கப்பட்டுள்ள மின்சாரங்கள் முயற்சி செய்யாது. விழுதாக்களில் பயன்படுத்தப்படும் ஒளிரும் தொடர்விளக்குகள் தொடர் இணைப்பில் இணைக்கப்பட்டிருக்கும். எடுத்துக்காட்டாக, மின்தடையாக்கிகள் வாடராக உள்ளடங்கிய ஒவ்வொரு மின்தடையாக்கியின் வழியாகவும் ஒரே அளவு மின்னோட்டம் பாயும்.

மின்சாரமேதானைது வீடுகள் மற்றும் தொழிற்சாலைகளுக்குப் பூமிக்கடியில் பதிக்கப்பட்ட கம்பி வடங்கள் அல்லது மன்மங்களை வரும் கம்பிகள் மூலம் அனுப்பப்படுகிறது. இப்பொருளானை ஒரு வீட்டு மின்சுற்று படம் 4.10 காட்டப்பட்டுள்ளது.

உருவாக்கப்படும் மின்சுற்றுகள் மூலமாக மின்சாரம் பகிர்ந்தளிக்கப்படுகிறது. மின்ஏற்றி மினபகிர்மைதானை செய்யும் இடத்திலிருந்து மினமைதாட்டமேதானைதும் முன்னணி மின்னளவிப் பட்டிக்குக் கொண்டுவரப்படுகிறது. முன்னணி மின்னளவிப் பட்டியில் இரண்டு முக்கியப் பாகங்கள் இருக்கும். (i) மின் உருகி இணைழ

மின்னளவிப் பட்டி எவ்வளவு மின்னறைல் பயன்படுத்தப்படுகிறது என்பதை அளவிடுகிறது. மின் உருகி இணைழ என்பது ஒரு சிறிய கம்பி இணைழ அல்லது ஒரு சிறிய மின்சுற்று உண்டாப்பி (MCB). வீட்டுப் பயன்பாட்டிற்கு மின்சாரங்களில் குறுக்கு மின்சுற்று

**படம் 4.10** வீட்டு மின்சுற்றுப் படம்

ஏற்படும்போது அதிகப்படியாக வரும் மினமைதாட்டத்திலிருந்து பாதுகாப்பு மின் உருகி இணைழ அல்லது சிறிய மின்சுற்று உண்டாப்பி மின்சுற்றை முறித்துவிடும். நீங்கள் மின் உருகி இணைழ பற்றி பகுதி 4.8.2யில் படித்துள்ளீர்கள். மின்சுற்று உண்டாப்பி என்பது தானைதாக மவதா அல்லது கமுனை உள்ளீடு மூலமாக மவதா தெயல்படுத்தக்கூடிய ஒரு தொழிற்சாலை ஆகும். இத் தொழிற்சாலையச் சுற்றிச் சிறிய கம்பிச் சுருள் சுற்றியிருக்கும். மின்சுற்றில் அதிகப்படியான மினமைதாட்டம் செல்லும்போது சுற்றியுள்ள கம்பிச் சுருள்தானைதும் மின்காந்தத்தால் ஈர்க்கப்படுகிறது. எனவே, மின்சுற்று உண்டாக்கப்பட்டு மின்சாரங்கள் பாதுகாக்கப்படுகின்றன. மின் உருகி இணைழ மற்றும் மின்சுற்று உண்டாப்பி ஆகியவற்றின் படம் 4.11ல்

![மின் உருகி இணைழ மற்றும் MCB](image)

**படம் 4.11** மின் உருகி இணைழ மற்றும் MCB

வீடுகளுக்கு வரும் மினமைதாட்டமேதானைது இரண்டு வி்ேதானை மின்காப்பிடப்பட்ட கம்பிகள் மூலமாகக் கொண்டுவரப்படுகின்றன. இம் இரண்டு கம்பிகளில் ஒன்று சிவப்பு காப்புனைக் கொண்ட கம்பி. அது மினமைதாட்ட கம்பி எனப்படும். கறுப்பு காப்புனை உள்ள வேறொரு கம்பி நடுநிலை கம்பி எனப்படும். நேதுவீட்டிற்குக் கொடுக்கப்படும் மின்சாரமேதானைது 220 வோல்ட் மின்னழுத்த வேறுபாடு கொண்ட ஒரு மாற்று மினமைதாட்டமே ஆகும். இவ்விரு கம்பிகளும் வாட்-மணி மீட்டருடன் (மின்னளவிப் பட்டி) இணைக்கப்பட்டுள்ளன. மினமைதாட்ட கம்பி மின் உருகி இணைழ வழியாக மின்னளவிப் பட்டியுடன்

இணைக்கப்பட்டுள்ளது. நடுநிலை கம்பி நேரடியாக மின்னளவிப் பட்டியுடன் இணைக்கப்பட்டுள்ளது.

மின்னளவிப் பட்டியிலிருந்து வரும் கம்பிதானைதும் முன்னணிச் தொழிற்சாலையுடன் இணைக்கப்பட்டுள்ளது. இத் தொழிற்சாலைதானைதம் அணைப்படும் போது மினமைதாட்டத்தை நிறுத்துவதற்குப் பயன்படுகிறது. முன்னணிச் சுற்றிலிருந்து வரும் மினமைதாட்ட கம்பிகள் வீட்டினுள் அணைக்கப்பட்டிருக்கும் னித் னிச் சுற்றுகளுக்குத் திணைணை வழங்கும். இருவகைதான மின்சுற்றுகள் வீடுகளில் பயன்படுத்தப்படுகின்றன. மின்பல்புகள், மின்விசிறிகள் அடங்கிய ஒரு சுற்றுக்கு 5 A அளவில்தானை குறைந்த திணை வழங்கும் சுற்றுகள் பயன்படுத்தப்படுகின்றன. குளிர்சாதனப் பெட்டிகள், நீர்சூமடற்றிகள், மின்எலணவப் பெட்டி, தராட்டிச் சுடும் அடுப்பு, மின்சார அடுப்பு, மின்சூமடற்றி, வெந்நீர்க் கொதிகலன் அடங்கிய மின்திணை சுற்றுகளுக்கு 15 A அளவில்தானை அதிக திணை வழங்கும் சுற்றுகள் பயன்படுத்தப்படுகின்றன. வீட்டிலுள்ள அணைத்து சுற்றுகளும் பக்க இணைப்பு முனையில் இணைக்கப்படுவதால் ஒரு சுற்றில் ஏற்பட்டாலும் அது மற்ற சுற்றுக்களுக்குப் பாதிப்பதாது. பக்க இணைப்பின் மற்றொரு நன்மை என்னைதவனில் அணைத்து மின்சாரங்களும் ஒரேதான மின்னழுத்தத்தின்

> **உங்களுக்குத் தெரியுமா?**
> இந்தியாவில் வீட்டுக்குறிய மின்சுற்றுகளில் 220/230 வோல்ட் மின்னழுத்தமும், 50 Hz அதிர்வெண்ணும் கொண்டு மினமைதாட்டம் அனுப்பப்படுகிறது. USA மற்றும் UK முதலிய நாடுகளில் வீட்டுக்குறிய மின்சுற்றுகளில் 110/120 வோல்ட் மின்னழுத்தமும் 60 Hz அதிர்வெண்ணும் கொண்டு மாற்று மினமைதாட்டம் அனுப்பப்படுகிறது.

## 4.10.1 அதிகப் பளுவாதல் மற்றும் குறுக்குத் மின்சுற்று
அதிகப் பளுவதால் அல்லது குறுக்குத் மின்சுற்று ஏற்பட்டதால் மின் உருகி இணைழ அல்லது சிறிய மின்சுற்று உண்டாப்பி மின்சுற்றை முறித்துவிடும். ஒரே மின்மூலதனில் அதிக அளவில்தானை மின்சுற்றுக்களுக்குத் தொடர் இணைப்பில் இணைக்கப்படும்போது அதிகப் பளு ஏற்படுகிறது. இது சுற்றின் வழியாக அதிகப்படியான மினமைதாட்டம் பாய்வதற்கு வழிவகை செய்கிறது. ஒரு மின்கம்பியில் அவ் எல்லா நிலையைத் தாண்டி அதிகப்படியான மினமைதாட்டம் பாயும்போது மின்கம்பி சூடாகி தீ ஏற்படுகிறது. இதுமவ, அதிகப் பளுவதால்

சில முறைகளில் வெப்பநிலை மாற்றம் அல்லது மற்ற காரணங்களால் மினமைதாட்ட கம்பியில் மாடப்பட்டுள்ள மின்காப்புனை பளுத்தாகிப் பாய்விடுகிறது. இம் காரணமாக மினமைதாட்ட கம்பிதானைது நடுநிலை கம்பியைத் தொடும் நிலை ஏற்படும். மினமைதாட்ட கம்பி நடுநிலை கம்பியுடன் தொடும்போது ஏற்படுவது தான குறுக்குத் மின்சுற்று. குறுக்குத் மின்சுற்று காரணமாக கம்பியின்

குறுக்குத் மின்சுற்று காரணமாக கம்பியின் மின்னடை மிகச் சிறியதாகிறது. இனைதால் அதிக அளவு மினமைதாட்டம் கம்பி வழியாகப் பாய்கிறது. இம் காரணமாக கம்பி சூடாகி தீ ஏற்பட்டு வீடுகளுக்குப்

## 4.10.2 புவித் தாடுப்பு
வீடுகளுக்கான மின்சுற்றில் பச்ணைக் காப்புனைத் தவிர மூன்றாவது கம்பி ஒன்று பயன்படுத்தப்பட்டிருக்கும். இக் கம்பியைப் புவித்தாடுப்புக் கம்பி என்று அழைப்பார்கள். புவித்தாடுப்புக் கம்பியின் மறுமுனையதானைது பூமியில் புதைக்கப்பட்ட உமைதாகக் குழாய் அல்லது உமைதாக் கடைகளுடன் இணைக்கப்பட்டிருக்கும். இக் கம்பிதானைதும் மினமைதாட்டத்திற்குக் குறைந்த மின்னடையைக் கொண்டிருக்கிறது. உமைதாகப் பரப்புடைய மின்எலணவப் பெட்டி, மேசை மின்விசிறி, குளிர்சாதனப் பெட்டி முதலிய மின்கருவிகளில் சில முறைகளில் மின்கசிவு ஏற்படும். மின்கசிவினால் உருவாகும் ஆபத்தான மினமைதாட்டம் புவித்தாடுப்புக் கம்பி வழியாகப் புவிக்குச் செல்கிறது. எனவே, புவித்தாடுப்புக் கம்பி இணைப்பதானைது ஒரு பாதுகாப்பு அரணாக அணைந்து மின்கசிவினால் உண்டாகும் மின்னழுத்தத்தைச் சீர்செய்கிறது.

## 4.11 LED மின்விளக்கு
LED பல்பு என்பது மின்சாரம் செல்லும்போது கண்ணுறு ஒளியை உமிழக்கூடிய ஒரு குணைக் கடத்திச் சாதனமே ஆகும். உமிழப்படும் ஒளியின் வண்ணம் பயன்படுத்தப்படும் பொருளின் தன்மையைப் பொறுத்து அமையும். சிவப்பு, பச்ணை, மஞ்சள் மற்றும் ஆரஞ்சு வண்ணங்களைக் கொண்டிருக்கும் LED பல்புகள் காலியம் ஆர்சனைடு மற்றும் காலியம் பாஸ்பைடு முதலிய மவதிச் சேர்மங்களைப் பயன்படுத்தி உருவாக்குகிறார்கள். டிஜிட்டல் கடிகாரங்கள், கைக்கீட்டுக் கருவிகள், பொருட்வரத்து எமிக்கைகள், திருவிளக்குகள், அலங்கார விளக்குகள் முதலியவற்றில் LED

## 4.11.1 ஏழுதுண்டுக் காட்சி
LED ஒளிஉமிழ் டையோடின் மற்றொரு முக்கியப் பயன்பாடு LED திரைக்காட்சி ஆகும். LED திரைக்காட்சி உண்மையில் ஒளிஉமிழ் டையோடு பயன்படுத்தப்பட்ட LCD (Liquid Crystal Display) திரைலக் காட்சி ஆகும். ## தீர்க்கப்பட்ட கணக்குகள்

**கணக்கு 1:** மின்னழுத்த வேறுபாடு முறையே 60 W, 220 V மற்றும் 40 W, 220 V. இரண்டில் எந்த விளக்கு அதிக மின்னடையைப் பற்றி இருக்கும்?

**தீர்வு:**
{{< katex display=true >}}
(P = \frac{V^2}{R}) {{< /katex >}}

மின்விளக்குகளிலும் ஒரே மதிப்பு உடையதாக இருப்பதால் மின்திணை மின்னடைக்கு எதிரவிகிதத்தில் இருக்கிறது.

40 W மின்விளக்குக்கு அதிக மின்னடை இருக்கும்.

**கணக்கு 2:** V மின்விளக்கில் பாயும் மினமைதாட்டம் மற்றும் மின்னடையைக் கைக்கிடு.

**தீர்வு:**
மின்திணை P = 100 W
மின்னழுத்தம் V = 200 V
மின்திணை P = VI எனவே, மினமைதாட்டம், {{< katex display=true >}}
I = \frac{P}{V} = \frac{100}{200} = 0.5 A{{< /katex >}}

மின்னடை, {{< katex display=true >}}
R = \frac{V}{I} = \frac{200}{0.5} = 400 Ω {{< /katex >}} 

**கணக்கு 3:** கீழே காட்டப்பட்டுள்ள மின்சுற்றில் 5 Ω, 10 Ω மற்றும் 20 Ω மின்னடை உடைய R₁, R₂ மற்றும் R₃ ஆகிய மூன்று மின்னடையாக்கிகள் இணைக்கப்பட்டுள்ளன.

![மின்சுற்று](image)

**தீர்வு:**
அ) மூன்று மின்னடையாக்கிகளும் பக்க இணைப்பில் இணைக்கப்பட்டுள்ளதால் மூன்று மின்னடையாக்கிகளுக்கும் எதிராக உள்ள மின்னழுத்தமும் ஒரேதாக இருக்கும். (i.e. V = 10V)

எனவே R₁ வழியாகச் செல்லும் மினமைதாட்டம் {{< katex display=true >}}
I_1 = \frac{V}{R_1} = \frac{10}{5} = 2A {{< /katex >}} 

R₂ வழியாகச் செல்லும் மினமைதாட்டம் {{< katex display=true >}}
I_2 = \frac{V}{R_2} = \frac{10}{10} = 1A {{< /katex >}} 

R₃ வழியாகச் செல்லும் மினமைதாட்டம் {{< katex display=true >}}
I_3 = \frac{V}{R_3} = \frac{10}{20} = 0.5A {{< /katex >}} 

ஆ) மின்சுற்றில் பாயும் மொத்த மினமைதாட்டம், {{< katex display=true >}}
I = I_1 + I_2 + I_3 = 2 + 1 + 0.5 = 3.5A {{< /katex >}} 

இ) மின்சுற்றில் உள்ள மொத்த மின்னடை:
{{< katex display=true >}}
\frac{1}{R_P} = \frac{1}{5} + \frac{1}{10} + \frac{1}{20} = \frac{4+2+1}{20} = \frac{7}{20} {{< /katex >}}

எனவே, {{< katex display=true >}}
R_P = \frac{20}{7} = 2.857 Ω {{< /katex >}}
